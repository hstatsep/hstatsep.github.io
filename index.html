A Few Words About SEP

In the Software Engineering Program, students do much more than simply write code.

Real Skills

Students learn to break down problems into smaller pieces. They learn to analyze, debug, collaborate on, and present their work. These are skills that they can take into any profession.

Real Tools

Students use real tools like the browser-based Cloud 9 IDE and Github to manage their code.

Real Fun

Students love this program because they get to use what they learn and apply it in a creative way. Students are proud of their projects, and that makes us proud of them.

Topics

What students learn each year

9: Scratch

Computational thinking through block-based programming of animations and games.

9: Robotics

Computational thinking through sensors and motors that interact with the physical world.

9: Python

Text-based code in a language common among high-level developers.

10: p5js

Computer programming fundamentals through a visual context.

10: Circuits

Electrical circuits through Makey Makey and Arduino.

10: Data

The manipulation of values with Google Sheets.

11: Github

Version control and collaborative software.

11: Web Design

Responsive, interactive websites through Bootstrap and jQuery.

11: 3D Modeling

3D renderings of objects.

12: Ruby

An object-oriented language that serves the back end of many web applications.

12: Sinatra

The Model-View-Controller framework and interaction with APIs.

12: Independent Study

Researching, building, and presenting on the topic of students' choosing.
